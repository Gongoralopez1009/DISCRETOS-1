Modelos Discretos de Probabilidad

¿Qué son los modelos discretos de probabilidad?

Primero, iniciemos con la definición de Distribución de probabilidad que implica que todos los posibles valores que resultan de un experimento aleatorio estén junto con la probabilidad asociada a cada valor.

Los modelos discretos de probabilidad son funciones matemáticas que asignan una probabilidad a cada valor posible de una variable aleatoria discreta. Para entenderlo de forma sencilla: una variable es discreta cuando sus resultados se pueden contar uno por uno (1, 2, 3…) y no admiten valores intermedios. Por ejemplo, puedes tener 2 hijos o 3 hijos, pero no 2.5.

Para que un modelo sea válido, debe cumplir dos reglas:

Probabilidad individual: Cada valor de probabilidad P(x) debe estar entre 0 y 1. Suma total: La suma de todas las probabilidades de todos los eventos posibles debe ser exactamente 1. Fórmula: Suma de P(xi) = 1.

2 .Modelo Binomial

\textbf{Distribución Binomial} \\ \text{Función de probabilidad: } P(X = x) = \binom{n}{x} p^x q^{n-x} \\ \text{Media: } \mu = n \cdot p \\ \text{Varianza: } \sigma^2 = n \cdot p \cdot q \\ \text{Desviación estándar: } \sigma = \sqrt{n \cdot p \cdot q}

n = Número de ensayos

X = Número de éxitos

p = probabilidad de éxito en cada ensayo

q = probabilidad de fracaso en cada ensayo

Se aplica a procesos donde se repite un número fijo de ensayos independientes y cada uno solo tiene dos resultados posibles.

Características:

  • El experimento consiste en n ensayos idénticos.

  • Cada ensayo tiene dos resultados: Éxito (p) o Fracaso (q).

  • La probabilidad de éxito p es constante en cada ensayo.

  • Los ensayos son independientes entre sí.

Ejemplo 1: Modelo Binomial

Contexto: Auditoría de Contratos en la Administración Pública.

Situación: Un auditor de la Contraloría sabe que el 10% de los contratos de obras menores tienen errores (p = 0.10). Se selecciona una muestra de 20 contratos (n = 20).

Pregunta: ¿Cuál es la probabilidad de hallar exactamente 3 contratos con errores (x = 3)?

Datos: n = 20 , p = 0.10 , q = 0.90 (Probabilidad de que no tengan errores)

Parametro Valor Descripcion
n 20.0 Número total de contratos (ensayos)

Función de Probabilidad Aplicada:

P(x = 3) = [ 20! / (3! * (20 - 3)!) ] * (0.10)^3 * (0.90)^17

P(3) = [ 20! / (3! * 17!) ] * 0.001 * 0.1667

P(3) = 1,140 * 0.001 * 0.1667

P(3) = 0.1901 (19.01%)

Media (u) = n * p = 20 * 0.10 = 2 contratos esperados.

Varianza (v) = n * p * q = 20 * 0.10 * 0.90 = 1.80.

Desviación Estándar (s) = raíz (1.80) = 1.34.

3 .Modelo Poisson

P(X = x) = \frac{e^{-\lambda} \lambda^x}{x!}; \quad \mu = \lambda; \quad \sigma^2 = \lambda; \quad \sigma = \sqrt{\lambda}

\lambda: Media o promedio de éxitos por unidad de medida.

x: Cantidad de éxitos cuya probabilidad se quiere hallar.

e: Constante de Euler (2.71828\dots).

!: Operación factorial.

Es una distribución de probabilidad discreta que se utiliza para modelar el número de eventos que ocurren en un intervalo fijo de tiempo, distancia, área o volumen, bajo el supuesto de que estos eventos ocurren con una tasa media constante y de forma independiente entre sí. Es ideal para situaciones donde la probabilidad de éxito es pequeña y el número de observaciones es muy grande (eventos raros).

Características:

  • La probabilidad de que ocurra un evento es proporcional al tamaño del intervalo.

  • Los eventos ocurren de forma independiente.

  • Se utiliza para eventos “raros” o conteos en periodos específicos.

4 .Modelo Hipergeométrico

\\ \text{Función de probabilidad: } P(X = x) = \frac{\binom{k}{x} \binom{N-k}{n-x}}{\binom{N}{n}} \\ \text{Media: } \mu = n \left( \frac{k}{N} \right) \\ \text{Varianza: } \sigma^2 = n \left( \frac{k}{N} \right) \left( \frac{N-k}{N} \right) \left( \frac{N-n}{N-1} \right) \\ \text{Desviación estándar: } \sigma = \sqrt{n \left( \frac{k}{N} \right) \left( \frac{N-k}{N} \right) \left( \frac{N-n}{N-1} \right)}

N (N mayúscula): Es el Tamaño de la población. Representa el número total de elementos o individuos que existen en el conjunto completo antes de extraer la muestra.

k (o a veces K): Es el Número de éxitos en la población. Representa cuántos elementos dentro de la población total poseen la característica que estamos estudiando (ej. piezas defectuosas totales, empleados con título, etc.).

n (n minúscula): Es el Tamaño de la muestra. Representa la cantidad de elementos que extraemos de la población total para ser observados.

x (o k minúscula en algunos textos): Es el Número de éxitos en la muestra. Representa cuántos de los elementos que sacamos en nuestra muestra de tamaño n poseen la característica deseada.

La Distribución Hipergeométrica es una distribución de probabilidad discreta que se utiliza para calcular la probabilidad de obtener un número determinado de éxitos en una muestra, sin reemplazo, extraída de una población finita.

Características:

  • La población es finita (N).

  • La probabilidad de éxito cambia en cada selección porque no se devuelve el elemento extraído.

  • Se conoce el número total de éxitos en la población (k).
